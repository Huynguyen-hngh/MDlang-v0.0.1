Mdlang v 0.0.1
1. Giới thiệu
Mdlang là một ngôn ngữ lập trình phiên bản sơ khai được tạo nên từ sự tò mò của cha đẻ nó: Nguyễn Huy
MDlang hiện tại gồm 2 thư viện:
iostd.h: Thay thế iostream + alias các kiểu dữ liệu và cú pháp cơ bản.
mdmath.h: Thư viện toán học, các hàm số, hằng số, random, Fibonacci.
2. Cách dùng ngôn ngữ lập trình
Do MDlang dùng nền của C/C++ nên:
Bước 1:  Click vào Embarcadero_Dev-Cpp_6.3_TDM-GCC_9.2_Setup.exe để chạy file C++
Bước 2: Sau khi cài đặt xong thì xin chúc mừng: BẠN ĐÃ CÀI COMPILE CHO MDLANG THÀNH CÔNG!
3. Các cú pháp của Mdlang
I, Thư viện iostd.h
1. Cơ bản
| Alias                | Thay thế               |
| -------------------- | ---------------------- |
| `set iostd standard` | `using namespace std;` |
| `program`            | `int`                  |
| Alias      | Thay thế |
| ---------- | -------- |
| `println`  | `cout`   |
| `input`    | `cin`    |
| `linedone` | `endl`   |
Ví dụ
#include "iostd.h"
set iostd standard
program main() {
    str name;
    println << "Nhập tên: ";
    input >> name;
    println << "Chào " << name << linedone;
    return 0;
}

5. Fibonacci
integer fibo_n(integer a, integer b, integer n);
Tìm số Fibonacci thứ n trong khoảng [a,b].
Trả về -1 nếu không tìm thấy.
Ví dụ thuật toán:
#include "iostd.h"
#include "mathematics.h"
set iostd standard;
program main() {
    integer n = 5;
    println << "Fibonacci thứ 5 trong [0,100]: " << fibo_n(0, 100, n) << linedone;
    return 0;
}
4. Ghi chú
Thư viện Mdlang là sơ khai, chưa tối ưu hiệu năng với Fibonacci lớn hay số học bigint.
Bạn có thể dùng code C++ ngay trong Mdlang vì nó là file .cpp.
Có thể mở rộng mdmath.h cho: phân số, ma trận, số nguyên tố, v.v.
